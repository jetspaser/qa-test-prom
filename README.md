
Установка зависимостей:
pip install pytest-playwright
playwright install


Команды:
Зайти в папку: cd qa-test-prom
Вернуться на уровень выше: cd ..

Запуск Pytest для конкретного файла:
Чтобы не гонять все тесты проекта, используй путь к файлу после команды pytest: 


Полезные флаги (Шпаргалка):
--headed: чтобы увидеть браузер (по умолчанию Playwright в pytest запускает его в фоне/headless).
--slowmo 1000: замедлить действия робота на 1 секунду (чтобы ты успел понять, куда он кликает).
-s: выводить все print() в консоль (если ты добавил их для отладки).
-x: остановиться сразу после первой упавшей ошибки (удобно при дебаге).


Пример «боевой» команды для дебага:
pytest pytest tests/test_main_page.py --headed --slowmo 500 -v
